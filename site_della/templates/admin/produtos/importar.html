Importar produtos via planilha

Como funciona o formato Bling:
  1. No Bling: Produtos → Exportar → CSV (ou XLSX)
  2. O arquivo exportado tem colunas ID, Código, Descrição, Preço, Situação
  3. A Descrição segue o padrão: MODELO (COR) (TAMANHO) — ex: BODY GIU (AZUL MARINHO) (PP)
  4. O sistema detecta o formato automaticamente e cria produtos, cores e tamanhos
  5. O código HEX das cores é preenchido automaticamente para cores conhecidas
  6. Escolha a categoria que será usada para todos os produtos importados
